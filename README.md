# TCG Collection App

## Deskripsi Aplikasi

TCG Collection App adalah aplikasi mobile berbasis Flutter yang digunakan untuk mengelola koleksi kartu Trading Card Game (TCG), seperti kartu Pok√©mon.  
Aplikasi ini memungkinkan pengguna untuk menambahkan, melihat, mengedit, dan menghapus data kartu koleksi secara lokal.

---

## Fitur Aplikasi

1. **Tambah Data Kartu**
   - Input nama kartu
   - Input series
   - Input rarity
   - Input harga
   - Upload gambar kartu

2. **Menampilkan Daftar Kartu**
   - List seluruh kartu yang telah ditambahkan
   - Tampilan card dengan gambar, nama, rarity, dan harga

3. **Edit Data Kartu**
   - Mengubah informasi kartu yang sudah tersimpan

4. **Hapus Data Kartu**
   - Menghapus kartu dari koleksi

5. **Filter Berdasarkan Rarity**
   - Menampilkan kartu berdasarkan tingkat rarity

6. **Popup Preview**
   - Menampilkan detail kartu saat ditekan
